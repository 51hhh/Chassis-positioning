# STM32F407VET6硬件详细配置

## MCU规格

| 参数 | 规格 |
|------|------|
| 型号 | STM32F407VET6 |
| 内核 | ARM Cortex-M4F |
| 主频 | 168 MHz |
| Flash | 512 KB |
| SRAM | 192 KB |
| 封装 | LQFP100 |
| 工作电压 | 1.8V - 3.6V |
| I/O电压 | 5V tolerant |

## 外设配置

### 时钟树
- **HSE**: 25 MHz 外部晶振
- **PLL**: 168 MHz 系统时钟
- **APB1**: 42 MHz（SPI2, CAN1, UART1）
- **APB2**: 84 MHz（SPI1, TIM11/13/14）

### SPI配置

#### SPI1 - AS5048编码器0（X轴）
| 参数 | 配置 |
|------|------|
| 引脚 | PA5(SCK), PA6(MISO), PA7(MOSI), PA4(CS) |
| 时钟 | APB2 84MHz ÷ 16 = 5.25 MHz |
| 模式 | Mode 1 (CPOL=0, CPHA=1) |
| 数据位 | 16-bit |
| 采样频率 | 20 kHz |

#### SPI2 - AS5048编码器1（Y轴）
| 参数 | 配置 |
|------|------|
| 引脚 | PB13(SCK), PB14(MISO), PB15(MOSI), PB12(CS) |
| 时钟 | APB1 42MHz ÷ 16 = 2.625 MHz |
| 模式 | Mode 1 (CPOL=0, CPHA=1) |
| 数据位 | 16-bit |
| 采样频率 | 20 kHz |
| 备注 | 降速避免通信故障 |

### CAN总线

#### CAN1 - YIS130 IMU
| 参数 | 配置 |
|------|------|
| 引脚 | PA11(RX), PA12(TX) |

| 采样点 | 87.5% (BS1=11, BS2=2) |
| 模式 | Normal |

### UART串口

| 参数 | 配置 |

### 定时器

#### TIM11 - 编码器采样（主循环）

## 中断优先级

STM32使用4位优先级（16级），配置为Preemption Priority模式：

| 中断 | 抢占优先级 | 说明 |
|------|-----------|------|
| DMA2_Stream2/7 | 0 | UART DMA（最高） |
| CAN1_RX/TX | 1 | CAN接收/发送 |
| TIM11 | 3 | 20kHz采样主循环 |
| TIM13/14 | 3 | 定时器 |
| SPI1/2 | 禁用 | 轮询模式 |

## PCB布线建议

### SPI信号
- 差分走线，阻抗50Ω
- 线缆长度<20cm
- 避免与高频信号（CAN）并行

### CAN总线
- 双绞线
- 120Ω终端电阻（总线两端）
- 最大长度40m @ 500Kbps

### 电源
- 3.3V主电源，纹波<50mV
- MCU和传感器分别供电，共地
- 编码器电源加10μF+100nF去耦电容

### 磁编码器安装
- AS5048距离磁铁: 0.5-3mm
- 磁铁轴向对准传感器中心
- 避免附近有铁磁材料
